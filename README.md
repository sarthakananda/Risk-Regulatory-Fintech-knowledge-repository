# Risk-Regulatory-Fintech-knowledge-repository with Claude
An experiment: Can 26 years of industry experience, combined with AI, produce something genuinely useful for the next generation — faster than either could alone? I think the answer is yes. Here's what I learned.
I've spent a week on average per function working deliberately (Testing) a collaboration model:
→ I provided the domain expertise, the scope, the practitioner context, and the critical review
→ Claude (Anthropic) structured, drafted, formatted, and — crucially — was asked to critically review its own output
→ That review identified six material errors. I verified each one against primary BCBS sources before accepting the corrections.
A free repository for newbies and developers to understand the domain in simple terms. Its a work in progress
